Este projeto tem o objetivo de realizar um minitutorial das ferramentas de docker que serão utilizadas na equipe.

Estrutura do tutorial:
1. Comandos básicos do docker
2. Configuração de redes
3. Docker compose 
4. Docker e ROS 

Motivação

A nvidea TX2 é suportada até o linux Ubuntu 18.00, por isso, para o funcionamento da equipe está sendo usado docker.
Para fins de rápida utilização faz-se necessário esse breve tutorial.
